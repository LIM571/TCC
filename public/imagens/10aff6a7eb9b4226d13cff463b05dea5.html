POO Herança Array

Arrays são uma estrutura de dados fundamental tanto em Java quanto em JavaScript. Eles permitem armazenar múltiplos valores em uma única variável.

Arrays em JavaScript

Em JavaScript, arrays são dinâmicos e podem conter elementos de diferentes tipos. Arrays são indexados a partir de zero e podem ser manipulados usando diversos métodos incorporados. 


// Exemplo de array heterogêneo
let lista = [1, 'dois', 3, 'quatro'];
console.log(lista); // [1, 'dois', 3, 'quatro']

// Exemplo de array homogêneo
let vetor = [1, 2, 3, 4];
console.log(vetor); // [1, 2, 3, 4]

// Acessando elementos do array
let animais = ["Cachorro", "Gato", "Porco", "Cavalo"];
console.log(animais[0]); // Cachorro
console.log(animais[2]); // Porco

Arrays em Java

Em Java, arrays são de tamanho fixo e devem conter elementos do mesmo tipo. O tamanho do array é definido no momento da criação e não pode ser alterado. 


// Exemplo de array homogêneo
int[] vetor = {1, 2, 3, 4};
for (int i : vetor) {
    System.out.println(i); // 1, 2, 3, 4
}

// Exemplo de array heterogêneo (usando Object)
Object[] lista = {1, "dois", 3, "quatro"};
for (Object obj : lista) {
    System.out.println(obj); // 1, "dois", 3, "quatro"
}

// Acessando elementos do array
String[] animais = {"Cachorro", "Gato", "Porco", "Cavalo"};
System.out.println(animais[0]); // Cachorro
System.out.println(animais[2]); // Porco

Diferenças Principais

  • JavaScript: Arrays podem conter diferentes tipos de dados e são dinâmicos.
  • Java: Arrays têm tamanho fixo e contêm elementos do mesmo tipo.
  • JavaScript: Métodos úteis incluem push, pop, shift, unshift.
  • Java: Arrays não têm métodos dinâmicos, mas listas (`List`) oferecem métodos como add, remove, get.
